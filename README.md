# Installation

Download the Geckodrivers from this repository
https://github.com/mozilla/geckodriver/releases

* Linux

      tar -xvzf geckodriver*
      chmod +x geckodriver
      sudo mv geckodriver /usr/local/bin/
      
* Windows

extract the exe and run it to install it.
